# FIR_25KHz_BW_DDS_In

This filter was designed using a linear-phase finite impulse response architecture to achieve precise frequency-domain control with stable and predictable behavior. Throughout the video, we’ll examine the filter coefficients, frequency response, FFT analysis, passband and stopband characteristics, and real-time signal processing performance.

We’ll also demonstrate how the FIR output data is exported into Simulink for system-level verification and spectral analysis. By the end of this presentation, you’ll have a complete understanding of the filter design methodology, implementation flow, and practical DSP validation techniques.

Filter Design Requirements: 

Sampling frequency: 0.5 MHz 
Cutoff frequency: 12.5 kHz 
Stopband frequency: 20 kHz 
Stopband attenuation: 60 dB 
Ripple in pass band: 0.2 dB
